# BootTask
静默执行shell脚本，可编辑！


# 切换
如果想添加其它的命令可直接修改task.plist文件中的list字段

# root用户命令
使用sntp命令同步系统时间，如下：
```
sudo -S sntp -sS 182.92.12.11 << EOF
123
EOF
```
其中**123**是用户密码
